# Roshni

A brand prototype for a premium Canadian medical scrub label, designed in Hamilton, Ontario.

This repository hosts the storefront mockup. It is a static HTML/CSS/JS site with no backend — product data lives in JavaScript, cart state persists via `localStorage`, and checkout is mocked.

## View the site

https://rubaiyat6370.github.io/roshni-site/

## Pages

- `index.html` — home
- `shop.html` — full collection with filters and sorting
- `product.html` — product detail with colour / size selection
- `about.html` — founder story and brand pillars
- `ethics.html` — sourcing, certifications, transparent pricing breakdown
- `journal.html` — editorial index
- `cart.html` — bag review and checkout summary
- `contact.html` — contact form and FAQ

## Structure

- `assets/css/styles.css` — complete design system
- `assets/js/products.js` — product catalogue and SVG garment renderer
- `assets/js/cart.js` — cart logic and checkout summary
- `assets/js/main.js` — shared interactions (drawer, menu, accordion)

## Status

Design prototype. Brand name, product copy, imagery, and pricing are working drafts and may change. No payments are processed.
